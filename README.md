# TrueWiki

TrueWiki is a local, provenance-aware wiki compiler for heavily modded game
modpacks.

The first target is a Terraria/tModLoader modpack, but the project should stay
centered on the general problem: collecting source material, resolving conflicts
and overrides, preserving curation, and generating a browsable wiki that answers
what is true in a specific modpack.

## Project Structure

- `docs/product/` - project goal, scope, quality principles, and failure modes
- `docs/architecture/` - compiler pipeline and early interface contracts
- `docs/research/` - source research and Terraria page-kind taxonomy
- `docs/roadmap/` - implementation sequencing notes
